## Installation

 **Walk through installation**
 
`npm init`

`npm install mysql2`

`npm install sequelize`

`npm install dotenv`

Run the following command at the root of your project and answer the prompted questions:

`mysql -u root -p`

Enter your password then enster thse comands:

`source db/schema.sql`

`quit`

`npm run seed`
  
`npm start`
